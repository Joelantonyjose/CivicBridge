# 🌉 CivicBridge – Foundation v1.0

CivicBridge is a Flask-based web application that enables citizens to report civic issues such as potholes, water leakage, damaged streetlights, and other public infrastructure problems.

The goal of CivicBridge is to create a simple, transparent platform that bridges the gap between citizens and public authorities by making issue reporting and tracking more accessible.

> **Foundation v1.0** establishes the core functionality of CivicBridge and serves as the base for future enhancements.

---

## ✨ Features

- 📝 Report new civic issues
- 📋 View all reported issues
- ✏️ Edit pending reports
- 🗑️ Delete reports
- ✅ Mark issues as resolved
- 🔍 Search reports
- 📂 Filter reports by status (Pending / Resolved)
- ⚠️ Input validation
- 💬 Flash messages for user feedback
- 🛡️ Backend validation and defensive programming

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone <repository-url>
cd CivicBridge
```

### 2. Install Flask

```bash
pip install flask
```

### 3. Run the application

```bash
python app.py
```

### 4. Open your browser

Visit:

```text
http://127.0.0.1:5000
```
